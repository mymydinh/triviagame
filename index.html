Trivia Game


Welcome to this extremely fun trivia game!

You have 30 seconds to answer each question. Test your knowledge on random trivia questions about games. Do you have what it takes to get all 10 correct? Let's go!

These questions were taken off the internet with sources that may not be 100% so if you got a wrong but you think you got it right, that's just too bad. The game isn't set it for you, it was for me. Have fun~
